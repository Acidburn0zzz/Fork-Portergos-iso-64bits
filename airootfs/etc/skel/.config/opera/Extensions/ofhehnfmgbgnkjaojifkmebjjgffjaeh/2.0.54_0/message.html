Enhancer for YouTube™ is now available for Edge

Great news for Windows 10 users!

Thanks to all the nice people who have rated Enhancer for YouTube™ the Microsoft Edge Extensions Team has added it to the Windows Store! If you are using Windows 10 Creators Update (version 10.0.15063.413) you can install Enhancer for YouTube™ on Edge. To do so visit this page then click on the Get the app button. If you are not using the latest version of Windows 10 you can use the Windows 10 Update Assistant to update your system.
